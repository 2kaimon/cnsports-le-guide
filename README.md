# cnsports-le-guide

## 项目简介

`cnsports-le-guide` 是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面以静态 HTML 形式呈现，不依赖于任何特定域名或网站平台，便于长期保存和访问。

本仓库的主要目的是整理和存放与体育领域相关的各类信息页面，内容以独立、自包含的 HTML 文件形式存在，方便用户直接浏览或本地使用。

## 目录说明

```
.
├── pages/          # 存放所有独立 HTML 页面
│   ├── *.html
│   └── ...
├── assets/         # 页面引用的公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
├── index.html      # 仓库首页（可选）
└── README.md       # 本文件
```

### 主要目录

- **`pages/`**：所有独立 HTML 页面均存放于此。每个页面为一个完整 HTML 文档，可脱离网络环境直接打开使用。
- **`assets/`**：存放页面所需的公共资源文件，包括样式表、脚本和图片等。部分页面可能内嵌资源，不依赖此目录。

## 页面归档说明

- 每个 HTML 页面均为独立归档，内容完整，不依赖外部服务或动态后端。
- 归档内容主要围绕体育领域相关信息，涵盖赛事、规则、历史、数据等主题（具体内容请在各页面中查看）。
- 页面采用静态 HTML 编写，兼容主流浏览器，可在本地或任何静态托管服务上直接打开。

## 维护说明

- 本仓库由维护者不定期更新，新增或修正页面以完善归档内容。
- 提交 Pull Request 或 Issue 时，请确保内容与体育领域相关，并尽量保持 HTML 文件的独立性和简洁性。
- 不接受包含广告、推广链接或商业营销内容的页面提交。
- 如发现页面链接失效或内容错误，欢迎通过 Issue 反馈。
